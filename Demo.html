Types of CSS Styles

In Line CSS

Here we can give the style, which line do we need

(Note: Attributes are used to change the behavior the element)

* Here we can use style attribute
* Ex: style="color: blue;"
* EX: style="background-color: blue"
* Not only colors we give border radius and its height also
=> style="background-color: blue; text-align: center; border-radius: 20px;height: 25px;"
* background-color=> key
* blue => value

Internal CSS

* Here we can Use Style Tag in Head
* Ex: h4{ color: blue; }
* We have a problem while using this style, If we need to modify the specific heading or paragraph color it will change the all same tag in that page, So we need to give unique identity by using of "id" attribute
* Ex: #heading{ background-color: rgb(255, 98, 0); text-align: center; border-radius: 20px; height: 25px; }

(Note: CSS Selectors
* These are attributes

=>tag

=>id-- we can use # in Internal CSS

=>class-- we can use . in Internal CSS

=>name

* they are the CSS selectors which ae used to give identity of the headings)

External CSS

* Here we can access the file in the folder of extension of .css. In that file we given attributes and its values.
* We are given all the attributes of this External CSS way is recommendable.
